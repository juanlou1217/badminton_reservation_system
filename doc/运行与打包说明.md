# 运行与打包说明

## 1. 创建虚拟环境

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## 2. 配置数据库

```bash
cp .env.example .env
```

填写远程 MySQL 信息：

```text
DB_HOST=juanlou.top
DB_PORT=3306
DB_USER=juanlou
DB_PASSWORD=实际密码
DB_NAME=juanlou
```

## 3. 初始化数据库

```bash
python -c "from app.db import init_db; init_db()"
python scripts/init_admin.py
python scripts/seed_demo_data.py
```

## 4. 启动程序

```bash
python main.py
```

## 5. 打包为 exe

Windows 环境执行：

```powershell
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
pyinstaller -F -w main.py -n 体育馆羽毛球预约系统
```

输出文件：

```text
dist/体育馆羽毛球预约系统.exe
```
