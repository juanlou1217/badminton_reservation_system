# 运行与打包说明

## 1. 创建虚拟环境

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## 2. 配置数据库

```bash
cp .env.example .env
```

填写远程 MySQL 信息：

```text
DB_HOST=juanlou.top
DB_PORT=3306
DB_USER=juanlou
DB_PASSWORD=实际密码
DB_NAME=juanlou
```

## 3. 初始化数据库

```bash
python scripts/check_db_connection.py
python -c "from app.db import init_db; init_db()"
python scripts/init_admin.py
python scripts/seed_demo_data.py
```

也可以使用一个 SQL 文件初始化表结构和演示数据：

```bash
mysql -h juanlou.top -P 3306 -u juanlou -p juanlou < sql/init.sql
```

`sql/init.sql` 不写入管理员密码。管理员账号仍建议使用 `python scripts/init_admin.py` 创建。

## 4. 启动程序

```bash
python main.py
```

## 5. 可选远程 MySQL 集成测试

```bash
RUN_MYSQL_TESTS=1 pytest tests/test_mysql_connection_optional.py -q
```

## 6. 打包为 exe

Windows 环境执行：

```powershell
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
pyinstaller -F -w main.py -n 体育馆羽毛球预约系统
```

输出文件：

```text
dist/体育馆羽毛球预约系统.exe
```
