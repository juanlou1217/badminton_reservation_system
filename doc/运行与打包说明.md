# 运行与打包说明

## 1. 创建虚拟环境

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## 2. 配置数据库

```bash
cp .env.example .env
```

`.env` 放在项目根目录，和 `main.py` 同级，不提交到 Git。

填写远程 MySQL 信息：

```text
DB_HOST=juanlou.top
DB_PORT=3306
DB_USER=juanlou
DB_PASSWORD=实际密码
DB_NAME=juanlou
```

## 3. 初始化数据库

`sql/init.sql` 是课程演示的主初始化入口，会一次性创建表结构、演示账号、默认设置、场地和当天时间段。

```bash
mysql -h juanlou.top -P 3306 -u juanlou -p juanlou < sql/init.sql
```

演示账号：

```text
管理员：admin / admin123
普通用户：user01 / user123
普通用户：user02 / user123
普通用户：user03 / user123
```

以上账号仅用于课程演示，正式部署后请修改密码。

执行后检查数据库连接和关键约束：

```bash
python scripts/check_db_connection.py
```

## 4. 启动程序

```bash
python main.py
```

## 5. Harness 工程化检查

```bash
make doctor
make check
```

`make doctor` 检查 Python、虚拟环境、依赖、关键文件和数据库环境变量。`make check` 会继续运行本地测试和编译检查。

## 6. 可选远程 MySQL 集成测试

```bash
RUN_MYSQL_TESTS=1 pytest tests/test_mysql_connection_optional.py -q
```

该测试会检查远程库连通性、项目表是否存在，以及 `reservations.active_slot_id` 生成列和 `uq_reservations_active_slot` 唯一索引是否存在。

也可以使用统一命令：

```bash
make doctor-mysql
make test-mysql
```

## 7. 打包为 exe

Windows 环境执行：

```powershell
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
pyinstaller -F -w main.py -n 体育馆羽毛球预约系统
```

输出文件：

```text
dist/体育馆羽毛球预约系统.exe
```
